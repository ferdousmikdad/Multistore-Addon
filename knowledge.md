# FlexStock Multi-Store Sync: Knowledge Base

---

## What is FlexStock Multi-Store Sync?

A powerful add-on that allows you to run multiple WooCommerce stores from a single Google Sheet.

- Sync shared stock across a network of stores, or keep each store completely independent
- Centralizes operations into one spreadsheet-driven workflow
- Eliminates the need to log in and out of different WordPress dashboards

---

## Why Use Multi-Store Sync?

### Without the Add-on (The Messy Way)
- Logging into stores one by one to make updates
- Repeating the same stock and product edits across multiple dashboards
- Manually trying to keep shared stock aligned, leading to inventory mistakes
- Wasting hours moving products from store to store

### With FlexStock Multi-Store Sync (The Managed Way)
- Choose exactly how each store behaves from one central hub
- Sync shared stock instantly where needed
- Keep separate stores fully independent without mixing up product data
- Reduce manual work, eliminate inventory mistakes, and scale without the chaos

---

## Store Setup Modes

### 1. Shared Stock Across Stores
For stores that should stay in sync.
- Update product data once and keep matching SKUs perfectly aligned across all connected stores
- **Best for:** Regional stores, Wholesale + Retail setups, branch locations, duplicate storefronts

### 2. Independent Inventory Mode
For stores that should stay separate.
- Manage multiple stores inside one Google Sheet while keeping each store's inventory fully separated
- **Best for:** Separate distinct brands, web agencies managing multiple client stores, multi-brand businesses

## Key Features

- **Real-Time + Scheduled Sync** — Push updates instantly or set them to run on your own schedule
- **Per-Store Tabs** — Every connected store gets its own clean tab in Google Sheets
- **Bulk Editing from Sheets** — Edit stock levels, prices, SKUs, and descriptions in bulk without opening WordPress
- **Two-Way Sync** — Keep WooCommerce stores and Google Sheets communicating in both directions
- **Add New Products** — Create and push entirely new products to connected stores from the spreadsheet
- **Formula Support** — Use native Google Sheets formulas for dynamic pricing, margin calculations, and inventory logic
- **Flexible Store Behavior** — Mix and match which stores sync together and which stay independent

## Frequently Asked Questions

**Can I sync the same SKU across multiple stores?**
Yes. Shared Stock mode is explicitly built for stores that share SKUs and need to stay perfectly aligned.

**Can I manage multiple stores without syncing their inventory together?**
Yes. Independent Inventory mode keeps store inventories entirely separate while still letting you manage them from one workflow.

**Can I use this for completely different brands?**
Yes. Managing separate brands without crossing data streams is one of the strongest use cases for this add-on.

**Do I always need to use Google Sheets?**
Not always. While Google Sheets powers bulk management, some basic store connections can be handled without relying on the spreadsheet as the main interface.

**Can I move products between stores?**
Yes. Import/export capabilities and store-to-store product movement are fully supported.

**How is this different from a normal multistore plugin?**
Most multistore plugins just connect databases. FlexStock Multi-Store Sync is built around store behavior and spreadsheet control, giving you unparalleled bulk-editing power and customized synchronization logic.
